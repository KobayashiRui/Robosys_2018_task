## 使用方法
```
sudo make
sudo insmod myled.ko
sudo chmod 666 /dev/myled0
echo [コマンド] > /dev/myled0
```
### コマンドの種類
+ a : 0.3秒間隔で4つのLEDが同時に10回点滅後すべてののLEDが点灯する
+ b : 0.3秒の速度でLEDが赤→黄→緑→青→赤の順番で10回点灯後すべてののLEDが点灯する
+ c : 0.3秒の速度でLEDが青→緑→黄→赤→青の順番で10回点灯後すべてののLEDが点灯する
